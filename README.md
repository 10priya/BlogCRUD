### API Endpoints

* **/createUser** (Create user or User registration)
* **/login** (User login)
 * **/createBlog** (Create Blog)
 * **/findallBlog** (get all Blogs)
 * **/updateBlog** (Update Blog by id)
 * **/deleteBlog** (Delete Blog by id)
